Python Quiz

1. What is the correct way to comment a single line in Python?

a) // This is a comment
b) # This is a comment
c) /* This is a comment */
d) -- This is a comment

2. Which of the following is used to define a function in Python?

a) func
b) define
c) def
d) function

3. What is the output of print(2 ** 3)?

a) 6
b) 8
c) 9
d) 16

4. Which of the following is used to open a file named "example.txt" in read mode?

a) file = open("example.txt", "r")
b) file = open("example.txt", "w")
c) file = read("example.txt")
d) file = "example.txt".open()

5. What does the len() function do?

a) Returns the largest item in an iterable
b) Returns the smallest item in an iterable
c) Returns the length of an object
d) Returns the type of an object

6. Which of the following is the correct way to create a list in Python?

a) list = (1, 2, 3)
b) list = [1, 2, 3]
c) list = {1, 2, 3}
d) list = <1, 2, 3>

7. What is the output of print("Hello" + "World")?

a) Hello World
b) HelloWorld
c) Error
d) Hello + World

8. Which of the following is used to get user input in Python?

a) get()
b) input()
c) read()
d) scanf()

9. What is the correct way to create a tuple in Python?

a) tuple = [1, 2, 3]
b) tuple = {1, 2, 3}
c) tuple = (1, 2, 3)
d) tuple = <1, 2, 3>

10. Which of the following is used to import a module in Python?

a) include
b) import
c) use
d) require

11. What is the output of print(10 / 3)?

a) 3
b) 3.0
c) 3.33
d) 3.3333333333333335

12. Which of the following is used to exit a loop prematurely?

a) exit
b) stop
c) break
d) return

13. What is the correct way to create a dictionary in Python?

a) dict = [key1:value1, key2:value2]
b) dict = (key1:value1, key2:value2)
c) dict = {key1:value1, key2:value2}
d) dict = <key1:value1, key2:value2>

14. Which of the following is used to define a class in Python?

a) def
b) class
c) struct
d) define

15. What is the output of print(bool(0))?

a) True
b) False
c) 0
d) Error

16. Which of the following is used to handle exceptions in Python?

a) try-catch
b) try-except
c) if-else
d) for-in

17. What is the output of print(type([]))?

a) list
b) array
c) tuple
d) set

18. Which of the following is used to remove whitespace from the beginning and end of a string?

a) strip()
b) trim()
c) remove()
d) clean()

19. What does the 'in' operator do in Python?

a) Performs matrix multiplication
b) Checks if a value exists in a sequence
c) Defines input variables
d) Creates a new instance of a class

20. Which of the following is used to round a number to the nearest integer in Python?

a) round()
b) ceil()
c) floor()
d) int()

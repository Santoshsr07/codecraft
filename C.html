C Programming Basics

Introduction to C

C is a general-purpose programming language created in the 1970s. It's known for its efficiency and is widely used in system programming, embedded systems, and application development.

Basic Program Structure


#include <stdio.h>

int main() {
    // Your code here
    printf("Hello, World!\n");
    return 0;
}

This is the basic structure of a C program. It includes a main function, which is the entry point of the program.

Certainly. Let's break down this basic C program structure: 1. #include
- This is a preprocessor directive that includes the standard input/output library (stdio.h). - It allows you to use functions like `printf()` for output.

2. int main() { ... }
- This declares the main function, which is the entry point of every C program. - `int` indicates that the function will return an integer value.

3. // Your code here
- This is a comment. Comments are ignored by the compiler and are used to explain the code.

4. printf("Hello, World!\n");
- This is a function call to 'printf()'', which prints text to the console. - "Hello, World!\n" is the string to be printed. The '\n' represents a newline character.

5. return 0;
- This statement returns 0 to the operating system, indicating that the program executed successfully.

6. The curly braces '{ }'' enclose the body of the main function.

This structure is the foundation for most C programs. You would typically add your own code between the opening and closing braces of the main function.

Data Types

  • int: Integer values
  • float: Single-precision floating point
  • double: Double-precision floating point
  • char: Single character

Variables

Variables are used to store data. In C, you must declare a variable's type before using it.


int age = 25;
float height = 1.75;
char grade = 'A';

Operators

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=, /=, %=

Control Flow

If-Else Statement


if (condition) {
    // code if condition is true
} else {
    // code if condition is false
}

For Loop


for (int i = 0; i < 5; i++) {
    // code to repeat
}

While Loop


while (condition) {
    // code to repeat while condition is true
}
